## Overview

This repository powers the public profile for the [Allure Framework](https://github.com/allure-framework) organization and stores default community health files for Allure repositories.

GitHub uses files from this repository as organization-wide defaults when a project does not provide its own copy. Repository-specific files still take precedence.

## Repository files

- [profile/README.md](profile/README.md) - the organization profile shown on GitHub.
- [LICENSE](LICENSE) - Apache License 2.0 for this repository.
- [CONTRIBUTING.md](CONTRIBUTING.md) - default contribution guidance.
- [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) - community expectations and enforcement contacts.
- [SECURITY.md](SECURITY.md) - responsible disclosure guidance.
- [SUPPORT.md](SUPPORT.md) - community support channels.
- [.editorconfig](.editorconfig) - baseline editor settings.
- [.gitignore](.gitignore) - common local files to exclude from version control.

## Updating the profile

Keep organization-wide guidance broadly applicable across Allure Framework projects. If a change only applies to one integration or product, prefer updating that repository directly.
