# Contributing to Allure Framework

Thank you for your interest in improving Allure Framework projects.

This repository contains the Allure Framework organization profile and default community health files. If your change is specific to one project, open the issue or pull request in that project's repository instead.

## Before you start

* Search existing issues and discussions before opening a new one.
* Use [Questions and Support](https://github.com/orgs/allure-framework/discussions/categories/questions-support) for usage questions.
* Use [General Discussion](https://github.com/orgs/allure-framework/discussions/categories/general-discussion) for ideas and broader topics.
* Do not report security vulnerabilities in public issues. Follow [SECURITY.md](SECURITY.md).

## Pull requests

For changes to this repository:

1. Fork the repository.
2. Create a focused branch for your change.
3. Keep Markdown changes concise and broadly applicable to the organization.
4. Check that all links point to the canonical Allure or Qameta destinations.
5. Open a pull request with a clear summary and any relevant context.

Maintainers may ask you to move repository-specific guidance into the affected project if it does not belong in this organization-wide default set.
