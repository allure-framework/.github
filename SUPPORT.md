# Support

Use these channels for Allure Framework help and community discussion:

* [Documentation](https://allurereport.org/docs/) - official Allure Report documentation.
* [Questions and Support](https://github.com/orgs/allure-framework/discussions/categories/questions-support) - usage questions and troubleshooting.
* [Official announcements](https://github.com/orgs/allure-framework/discussions/categories/announcements) - releases and project updates.
* [General Discussion](https://github.com/orgs/allure-framework/discussions/categories/general-discussion) - ideas, feedback, and community conversation.

For reproducible bugs or feature requests, open an issue in the repository that owns the affected package or integration.

Do not share security vulnerabilities in public support channels. Follow [SECURITY.md](SECURITY.md).
